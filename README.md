## Проект по A/B-тестировнию

### Входные данные: 
Датасет взят отсюда: https://www.kaggle.com/datasets/faviovaz/marketing-ab-testing?resource=download

Категории: AD - пользователи, видевшие рекламу. PSA - пользователи, видевшие "социальную" рекламу

### Цель проекта: 
Проверить, повысилась ли конверсия у компании после внедрения рекламы.

### Технологии: 
numpy, pandas, matplotlib, seaborn, scipy.stats.

### Методология: 
В ходе проекта были проведены:
1. Очистка данный: была проверка на пропуски, а так же очистка от аномальных значений для последующего увеличения точности теста (ограничение по 99 квантилю).
2. EDA: были просмотрены данные по различным категориям и сделаны выводы.
3. Проверка гипотез: был проведен z-test для двух выборок и рассчитан MDE.

### Выводы
1. Было обнаружено, что плато конверсий возникает при 120-140 показах, значит можно ограничить число показов данными значениями, что сэкономит затраты на рекламу и может повысить прибыль.
2. Было обнаружено, что среди пользователей, больше всего конверсий приходилось на понедельник (25000+), в последующие дни конверсия падала и попадала в плато (около 16000)
3. Был обнаружен сильный дисбаланс между классами (класс AD в 24 раза больше класса PSA), z-test стабилизирует этот дисбаланс.
4. По итогам проведения тестов были получены значения p-value и MDE, получено статистически-значимое подтверждение повышения конверсии и достаточной чувствительности теста, превышающая разницу конверсий примерно в 3 раза, из чего следует статистическое доказательство повышения конверсии при внедрении рекламы.
